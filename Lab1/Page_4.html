Sida 4

on this page you make a written description of how to use the structural HTML5 elements

Bas implementation

Dessa taggar måste finnas med i varje websida

!Docktype html = Identifierar sidan som en html sida
html = Hyper Text Markup Language här startar själva websidan
head = Här skrivs allt som ska användas av webbrowsern (språk, nyckelord för sökmotorer, scripts med mera)
title = Det som skrivs här kommer att synas högst upp i browserns tablå, (denna sida har Sida 4 som title)
body = Här inne i body ska kommer allt som ska visas på skärmen upp. bilder, text, videor med mera

Vanliga implementationer

Dessa taggar är viktiga att kunna

p = paragraph Detta är standardiserad strötext som finns på websidan
b = bold Gör att texten blir fetmarkerad
i = italic Kursiverar din text
u = underline Stryker under din text
ul = unordered list En punktbeskriven lista
ol = ordered list En sifferbeskriven lista
li = list item Beskriver vad som ska finnas med i listan
dl = description list Skapar en med ord beskriven lista
dt = description term Vad den beskrivna listan ska innehålla
dd = description detail Tabbar in och detaljbeskriver saker i den beskrivna listan
hx = header(value) En huvudrubrik som med fördel skrivs först i sektioner eller artiklar. (finns i 6 format)
img = image insättningar av diverse bilder
strong = strong Definjerar ett eller flera ord av betydelse
a href=URL = a hyperlink reference Skapar hyperlänkar för att navigera runt på webben
abbr title="" = abbreviation Används vid exempelvis förkortningar för att vid hovring över ordet ge en beskrivning vad ordet betyder
meta = metadata Data som beskriver data
small = small gör det du skriver betydande smått (passar bra vid bland annat trademarks)
mark = marked När du vill markera en specifik text
time = time Setter tid
command = command Agerar som en metod (får kolla mer om detta)
